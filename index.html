JSON

@(语法)[方法|stringify|parse|toJSON]

JSON 是JavaScript的一个严格的子集，利用JavaScript中的一些模式来表示结构化数据。
JSON 只是一种数据格式，不是一种编程语言。很多编程语言都有针对JSON的解析器和序列化器。

语法

  • 简单值 ：使用与javascript相同的语法，可以在JSON中表示字符串、数值、布尔值和null。但JSON值不支持javascript中的特殊值undefined；
  • 对象 ： 对象作为一种复杂数据类型，表示的是一组无序的键值对。每个键值对中的值可以是简单值，也可以是复杂数据类型的值；
  • 数组 ：也是一种复杂数据类型，表示的是一组有序的值的列表，可以通过数组索引来访问其中的值。数组的值也可以是任意类型——简单值、对象或数组。

JSON对象有两个方法：
stringify（）把javascript对象序列化为JSON字符串;
parse（）把JSON字符串解析为原生javascript值。

stringify()方法默认情况下输出JSON字符串不包含任何空格字符或缩进。
所以返回的结果字符串为：
{“title”:”javscript”,”authors”:[“jkt”],”edition”:3,”year”: 2009 }

在序列化javascript对象时，所有函数及原型成员都会被有意忽略，不体现在结果中。而且，值为undefined的任何属性都会被跳过。

一、序列化选项：

JSON.stringify（）除了要序列化的javascript对象（第一个参数）外，还可接受另外两个参数。

第二个参数是过滤器，可以是一个数组或者函数。
如果是数组：结果中则只列出包含在数组中的的属性。

返回的结果字符串：{“title”:”javscript”,”edition”:3}
如果是函数：函数接收两个参数：属性（键）名、属性值 。

属性名只能是字符串，如果值不是键值对结构的值，键名可以是空字符串 。函数返回的值就是相应键的值，但如果函数返回的是undefined，那相应的属性会被忽略。

JSON.stringify()的第三个参数用于控制结果中的缩进和空白字符。
如果该参数是一个数值，表示每个级别缩进的空格数。

返回的结果：

{
    "title": "javascript",
    "authors": [
        "jkt"
    ],
    "edition": 3,
    "year": 2009
}

只要传入例如有效的控制缩进的参数值，结果字符串中就 会包含换行符 。最大缩进空格数为10，大于10会自动转换为10

如果该参数是一个字符串，则这个字符串会被用作缩进符。

返回的结果：

缩进字符串最长不能超过10个字符长，超出了则只显示前10个字符

返回undefined：
如果此时包含它的对象嵌入在另一个对象中，会导致他的值变成null；
如果它的对象是顶级对象，结果就是undefined。

很重要的序列化的内部顺序

假设把一个对象传入JSON.stringify（)，序列化该对象的顺序如下：
（1）如果存在toJSON（）方法而且能通过它取得有效的值，则调用该方法，否则返回对象本身；
（2）如果提供了第二个参数，应用这个函数过滤器。传入函数过滤器的值是第（1）步返回的值；
（3）对第二步返回的每个值进行相应地序列化；
（4）如果提供了第三个参数，执行相应的格式化。

二、解析选项：

JSON.parse（）方法也可以再接收一个参数，该参数是一个函数（还原函数），将在每个键值对上调用。与JSON.stringify()接收的函数（过滤函数）的签名是相通的——都接受两个参数：键、值，且都需要返回一个值。

如果还原函数返回undefined，则表示从结果中删除相应的键；如果返回其他值，则将该值插入到结果中

string

字符串操作方法

@(String类型)[操作方法|slice|substr|substring]

String类型 是字符串的对象包装类型。它的字符串操作方法主要有以下几种：

  • concat() ：用来连接一个或多个字符串，返回的是新字符串，不会改变原字符串；
  • slice() ： 基于子字符串创建新字符串，返回新字符串，也不会该变原字符串
  • substr() ：同第二个；
  • substring() ：同第二个。

slice()

该方法可以接受一个或两个参数，第一个参数指定字符串开始的位置，第二个参数指定到哪里结束。

substr()

该方法也可以接受一个或两个参数，但是第一个参数指定字符串开始的位置，第二个参数指定截取几个。

substring()

该方法也可以接受一个或两个参数，第一个参数指定字符串开始的位置，第二个参数指定结束位置。

数组和字符串的操作方法

字符串
属性：length
str=’javascript’;
alert(str.length); //8
方法：
indexOf();字符第一次出现的位置检索
str=’javascript’;
alert(str.indexOf(‘s’)); //4
lastIndexOf(); 字符最后一次出现位置检索
str=’javascript’;
alert(str.lastIndexOf(‘a’)); //2 最后一个a出现的位置
substr(0,5);字符串截取,从位置0截取到位置5=substring（0,5）
substring（1，5）：从1开始到5结束
substr(1,5) :从1开始取5个
str=’/web/home/inex.php’;
alert(str.substr(1,3)); //web
综合例子：截取路径中的目录部分
str=’/web/home/inex.php’;
pos=str.lastIndexOf(‘/‘);
dir=str.substr(0,pos);
alert(dir); /// web/home
截取文件部分
str=’/web/home/inex.php’;
pos=str.lastIndexOf(‘/‘);
file=str.substr(pos+1); //第二个不写，默认截取到最后一个
alert(file); //index.php
截取文件.后的后缀部分
方法一：
str=’/web/home/inex.php’;
pos=str.lastIndexOf(‘.’);
ext=str.substr(pos+1);
alert(ext); //php
方法二：
file=’inex.class.html.php’;
// 用.分割成四个数组
arr=file.split(‘.’);
l=arr.length;
// 取数组最后一个值即可
alert(arr[l-1]);
取数组最后一个值还有另外的方法：
val=arr.pop();
alert(val); //php

val=arr.slice(3);  //直接取第三个
alert(val);             //php

 arr2=arr.slice(-1);  //直接取第三个

alert(arr2); //php
alert(typeof(arr2)); //object
截取的结果还是数组，所以此处slice属于数组截取，要想截取到的是字符串
alert(arr2[0]); //截取的是字符串php

file='inex.class.html.php';
alert(file.slice(5,10));  //class 此处slice是字符串截取

val=arr.slice(1,3);  //从1开始，截取到3但不包括第三个
alert(val);                     //class,html

slice(start,end); 从哪开始到哪结束不包括最后一个位置
split(reg); 把字符串分隔成数组（也支持正则分割）
str=’2017-7-14’;
arr=str.split(‘-‘);
alert(arr); // 2017,7,14
search(reg); 字符串查找
str=’/web/home/index.php’;
pos=str.search(‘home’); // pos=str.indexOf(‘home’);有一样的效果
alert(pos); // 5

str=’/web/home/index.php’;
pos=str.search(/Home/i); //正则 ，忽略大小写
alert(pos);
match(reg); 字符串匹配，返回的是数组
str=’/web/home/index.home.php’;
arr=str.match(/home/g); //正则表达式 ，g全部匹配
alert(arr); //home,home

str=’/web/home/index.HOME.php’;
arr=str.match(/home/ig); //正则表达式
alert(arr); //home,HOME
replace(reg,’str’); 字符串替换，不动原来的值只改变新值
str=’/web/home/index.HOME.php’;
str.replace(‘HOME’,’home’);
alert(str); // /web/home/index.HOME.php

str=’/web/home/index.HOME.php’;
str2=str.replace(‘HOME’,’home’);
alert(str2); // /web/home/index.home.php

str=’/web/home/index.HOME.php’;
str2=str.replace(/HOME/ig,’jkt’);
alert(str2); // /web/jkt/index.jkt.php
